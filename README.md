Este projeto é um Order Management System desenvolvido como exercício de treinamento em programação orientada a objetos. O foco está na criação da arquitetura do sistema, definição de classes, métodos e relacionamentos entre entidades como Order, Customer e Product.

O objetivo é consolidar conceitos de herança, polimorfismo e boas práticas de desenvolvimento, preparando a base para futuras implementações de funcionalidades reais, como gerenciamento de pedidos, atualização de status e geração de relatórios.
